1. Рецепт
- id (PK)
- Название блюда
- Пользователь (FK)
- Заголовок
- Время приготовления (в часах)
- Список ингридиентов (строка, перечисление)
- Описание
- Фото блюда
- Рейтинг (автоматически формируется после добавления каждого отзыва)
- Кол-во порций
- Проверен ли
- Дата публикации (автоматическая)
- Ккал (на 100г, int)
- Белки (на 100г, double)
- Жиры (на 100г, double)
- Углеводы (на 100г, double)
2. Пользователь
- id (PK)
- Ник
- Логин
- Пароль
- Описание
- Фото
- Дата регистрации
- Дата рождения
- Рейтинг (средний по рецептам)
- Причина удаления (текст, nullable)
3. Этапы рецептов
- id (PK)
- Рецепт (FK)
- Номер этапа
- Фото
- Описание
4. Ингридиенты
- id (PK)
- Название
- Ккал (на 100г, int)
- Белки (на 100г, double)
- Жиры (на 100г, double)
- Углеводы (на 100г, double)
5. Ингридиенты рецепта
- id (PK)
- Ингридиент (FK)
- Рецепт (FK)
- Граммовка (int)
6. Теги
- id (PK)
- Название
- Описание
7. Теги рецепта
- id (PK)
- Тег (FK)
- Рецепт (FK)
8. Сообщение
- id (PK)
- Пользователь (Отправитель, FK)
- Второй пользователь (Получатель, FK)
- Текст
- Дата и время
9. Избранные рецепты
- id (PK)
- Рецепт (FK)
- Пользователь (FK)
10. Рецензия
- id (PK)
- Пользователь (FK)
- Оценка (1-5)
- Заголовок
- Текст
- Дата написания
- Причина удаления (nullable)
11. Друзья
- id (PK)
- Пользователь (Кто отправил заявку, FK)
- Второй пользователь (Кто принял заявку, FK)
12. Рецензии рецептов
- id (PK)
- Рецензия (FK)
- Рецепт (FK)
